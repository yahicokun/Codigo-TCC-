# Codigo-TCC-
Códigos utilizados no meu trabalho de conclusão de curso 

treinamento_rede_mlp

Configurações: 


JSON_PATH : Caminho do Json que faz a configuração inicial da rede e do descritor escolhido


DB_PATH = Caminho do banco de dados. É importante colocar um r antes do caminho para que ele saiba que é um arquivo para leitura


TARGET_KEY = Qual informação será extraída do banco de dados. Como o código está configura agora ele só funciona para o gap


LIMIT = None  Deixar sempre em none, as vezes o código buga se alterar esse valor 


gerador_json_descritores

Configurações: 

Apenas escolher qual descritor deseja criar. 

Importante!!! 

Adicionar o caminho correto do banco de dados no seu computador 

Ajustar para qual pasta os resultados devem ser salvos
